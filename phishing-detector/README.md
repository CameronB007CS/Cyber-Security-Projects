# PhishGuard 

A hybrid phishing detection Chrome extension built as part of a cybersecurity portfolio project.

## How It Works
PhishGuard uses a two-layer detection approach:
- **Heuristic engine** — rule-based checks on URLs and page content
- **ML model** — trained classifier to catch what the rules miss
